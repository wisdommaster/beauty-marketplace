## 1. Мини-карточка в ленте

### Структура

```
┌──────────────────────────────────────────────┐
│ 🔵 Вэлнес                                    │ ← цветной пин категории
│                                              │
│ Хатха-йога                                   │ ← конкретное направление
│                                              │
│ 🕐 18:00–19:30                               │ ← время начала
│                                              │
│ ~~2 400₽~~  1 680₽  −30%                    │ ← цена было → стало + % скидки
│                                              │
│ 📍 3.2 км  ул. Арбат, 12                    │ ← расстояние + адрес
│                                              │
│ ⭐ 4.8  Yoga Space Studio                     │ ← рейтинг + название студии
└──────────────────────────────────────────────┘
```

### Цветные пины по категориям

| Категория | Цвет | HEX |
|---|---|---|
| Вэлнес | 🔵 Синий | `#3B82F6` |
| Бьюти | 🟣 Фиолетовый | `#8B5CF6` |
| Спа | 🟢 Изумрудный | `#10B981` |
| Спорт | 🟠 Оранжевый | `#F97316` |

Пин — компактный цветной бейдж в верхней части карточки с названием категории.

### Состояния мини-карточки

| Состояние | Поведение |
|---|---|
| Доступно | Полноцветная, активная |
| Мало мест (≤2) | Бейдж «2 места» |
| Последнее место | Бейдж «Последнее» красный |
| Истекает (<2ч) | Иконка 🔥 |
| Забронировано | Серая, зачёркнутая, неактивная |
| Уже в избранном | Сердечко заполнено ❤️ |


**2.6 Маршрут (3 пина)**

```
┌─────────────────────────────────────────────┐
│  Как добраться?                             │
│                                             │
│  🚗 На авто    🚕 Такси    🚶 Пешком       │ ← 3 кнопки-пина
│     12 мин        14 мин       38 мин       │ ← расчётное время
│         ↗            ↗            ↗         │ ← стрелка = переход
└─────────────────────────────────────────────┘
```

При тапе на любой пин → модальное окно:

```
«Вы переходите в Яндекс.Карты для построения маршрута.
Способ: На авто (12 мин)»
                             [Отмена]  [Перейти]
```

После подтверждения → открытие Яндекс.Карты с построенным маршрутом.

**Техническая реализация:**
- Deeplink в Яндекс.Карты: `yandexmaps://maps.yandex.ru/?rtext=lat1,lon1~lat2,lon2&rtt=auto/pd/mt`
- `rtt=auto` — автомобиль, `rtt=pd` — пешком, `rtt=mt` — такси (через Яндекс.Go)
- Расчёт времени — через Яндекс.Маршрутизатор API (GET /routing)

## 3. Настройка тарифа бизнесом

### Концепт: бизнес управляет скидкой и радиусом в реальном времени

Бизнес видит превью охвата и принимает решение.

### Интерфейс (`/business/tariff/customize`)

**3.1 Ползунок радиуса**

```
┌─────────────────────────────────────────────┐
│  Радиус показа                              │
│  ○────────────────●─────────────○  10 км    │
│  1 км            5 км          10 км        │
│                                             │
│  В этом радиусе:                            │
│  👥 ~320 000 пользователей                  │
│  📍 ~45 студий-конкурентов                  │
│  💰 Средняя скидка в зоне: 22%              │
└─────────────────────────────────────────────┘
```

- Границы ползунка: 1 км – макс. тарифа (зависит от тарифа: Growth = 1-10 км, Maximum = 1-25 км)
- Реальное время обновления цифр при движении ползунка
- «Рекомендуемый радиус» — отметка на ползунке (автоматически: город = 7-10 км, спальный район = 3-5 км)

**3.2 Ползунок максимальной скидки**

```
┌─────────────────────────────────────────────┐
│  Максимальная скидка                        │
│  ○─────────●───────────────────○   35%      │
│  10%       25%                35%           │
│                                             │
│  Минимальная скидка (фикс.): 10%            │
│                                             │
│  При 35% максимальной скидке:               │
│  📊 Средний чек платформы: 3 200₽           │
│  📊 Конверсия пустых слотов: +24%            │
│  ⚠️  Маржа ↓ на 8% относительно 25%        │
└─────────────────────────────────────────────┘
```

- Min скидка — фиксирована (зависит от тарифа)
- Max скидка — регулируется в пределах [min+5%, тарифный максимум]
  - Growth: min=10%, max регулируется 15%-35%
  - Maximum: min=15%, max регулируется 20%-50%
- При движении ползунка обновляются: средний чек, конверсия, влияние на маржу

**3.3 Превью для конкретной услуги**

```
┌─────────────────────────────────────────────┐
│  Тайский массаж 60 мин — 3 500₽             │
│                                             │
│  При текущих настройках (10 км, 10-35%):    │
│                                             │
│  Расстояние │ Скидка │ Цена    │ Охват      │
│  ───────────┼────────┼─────────┼────────────│
│  0-2 км     │ 10%    │ 3 150₽  │ ~120 000   │
│  2-5 км     │ 18%    │ 2 870₽  │ ~90 000    │
│  5-8 км     │ 25%    │ 2 625₽  │ ~65 000    │
│  8-10 км    │ 35%    │ 2 275₽  │ ~45 000    │
│                                             │
│  Итого охват: ~320 000 пользователей         │
│  Средняя скидка: 19.5%                       │
└─────────────────────────────────────────────┘
```

### 3.4 Умные подсказки

| Ситуация | Подсказка |
|---|---|
| Бизнес поставил максимальный радиус | «Отлично! Вы охватываете весь доступный район. Ожидайте в 2× больше показов.» |
| Бизнес поставил минимальный радиус | «Узкий радиус = быстрая конверсия. Ближайшие клиенты бронируют в 3× чаще.» |
| Бизнес поднял скидку до максимума | «При максимальной скидке заполняемость пустых слотов растёт на ~40%. Но маржа ниже.» |
| Бизнес снизил скидку | «Меньше скидка — выше средний чек. Но конверсия может упасть на 10-15%.» |
| Радиус > средней зоны конкурентов | «Вы охватываете на 30% больше аудитории, чем студии рядом. Конкурентное преимущество.» |

### 3.5 История изменений

```
┌─────────────────────────────────────────────┐
│  История настроек                           │
│                                             │
│  12.04  Радиус: 7→10 км  Скидка: 30→35%    │
│  05.04  Радиус: 5→7 км   Скидка: 25→30%    │
│  28.03  Первоначальные настройки            │
└─────────────────────────────────────────────┘
```

### 3.6 Кнопка применения

```
[Сбросить]                    [Применить настройки]
```

При применении:
- Модальное окно: «Новые настройки применятся к текущим и будущим слотам в течение 5 минут. Существующие брони не затрагиваются. Продолжить?»
- После подтверждения → мгновенное обновление в БД маркетплейса
- Следующая синхронизация с YClients подхватит новые параметры

## 4. Раскрытие «Почему такая цена?» — единый блок

Для мини-карточки (свёрнуто):
```
−30%  💡
```

При тапе на 💡 → попап:
```
Базовая цена      3 500₽
Расстояние        6 км из 10 км
Скидка           10% (база) + 11% (за расстояние) = 21%
Вы экономите      735₽
```

Для раскрытой карточки — блок развёрнут всегда.

# Beauty Marketplace — Финальный концепт

## Что это

Маркетплейс для beauty/wellness-бизнесов. Помогает заполнять пустые слоты в расписании через радиусную динамическую скидку. Бизнес публикует свободный слот → пользователи в радиусе видят его со скидкой → чем дальше пользователь, тем больше скидка.

---

## Ключевые решения

### Формула скидки
```
discount = бизнес_min + (distance / бизнес_radius) × (бизнес_max - бизнес_min)
```
- Считается только от бизнес-тарифа
- Пользовательский радиус — только фильтр видимости
- На одном расстоянии все тарифы получают одинаковую скидку

### Тарифы
**Бизнес:** Start (0₽, 3км, 5-20%, 7 слотов/нед) / Growth (3 990₽, 10км, 10-35%, 14 слотов/нед) / Maximum (7 990₽, 25км, 15-50%, 21 слотов/нед)
**Пользователь:** Free (0₽, 7км, 3 рефреша) / Comfort (299₽, 20км, 10 рефрешей) / Premium (599₽, ∞, безлимит)

### Временное окно: ≤12ч для всех тарифов

### 1 публикация = 1 тайм-слот (групповое 0/4 = 1 карточка)
### Сброс лимита: каждый понедельник 00:00. Не использовал — сгорело

### YClients: downstream-синк. Платформа — source of truth для пользователя
### Монетизация: только тарифы, 0% комиссии с транзакций
### Рейтинг: Яндекс.Карты API (1-5 звёзд)
### Оплата: два режима — OFFLINE_ONLY / ONLINE_REQUIRED (YooKassa, 100% бизнесу)
### Таймауты: 5 мин (онлайн-оплата), 10 мин (офлайн-подтверждение)
### Отмена: 100% возврат >30 мин, 0% ≤30 мин
### No-show: 3 инцидента за 30 дней → ограничение

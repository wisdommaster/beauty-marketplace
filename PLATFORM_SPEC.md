## 3. Регистрация

### 3.1 Выбор роли (`/register`)

Две карточки:
- «Я клиент» → `/register/user`
- «Я бизнес» → `/register/business`
- Ссылка «Уже есть аккаунт? Войти →» `/login`

### 3.2 Регистрация пользователя (`/register/user`)

| Шаг | Поля | Валидация |
|---|---|---|
| 1 | Имя, Фамилия | Обязательно, ≥2 символа |
| 2 | Телефон (+7) | Обязательно, формат, проверка SMS-кодом |
| 3 | Email | Опционально |
| 4 | Пароль | ≥8 символов, цифра + буква |
| — | Чекбокс «Принимаю условия» | Обязательно |

Кнопка: «Зарегистрироваться» → редирект на `/onboarding/user`

### 3.3 Регистрация бизнеса (`/register/business`)

| Шаг | Поля | Валидация |
|---|---|---|
| 1 | Название студии | Обязательно |
| 2 | Категория (выпадающий список) | Обязательно |
| 3 | Адрес | Обязательно + карта |
| 4 | Имя владельца/администратора | Обязательно |
| 5 | Телефон | Обязательно, SMS-проверка |
| 6 | Email | Обязательно |
| 7 | Пароль | ≥8 символов |

Кнопка: «Зарегистрироваться» → редирект на `/onboarding/business`

### 3.4 Вход (`/login`)

| Поле | Тип |
|---|---|
| Телефон или Email | input |
| Пароль | input (password) |

- Кнопка «Войти»
- Ссылка «Забыли пароль?»
- Ссылка «Нет аккаунта? Зарегистрироваться →»

## 4. Онбординг

### 4.1 Онбординг пользователя (`/onboarding/user`)

3 шага, обязательно все. Прогресс-бар сверху.

| Шаг | Название | Элементы |
|---|---|---|
| 1 | **Что ищешь?** | Мультивыбор категорий (сетка иконок): Массаж / Йога / Косметолог / Спа / Парикмахер / Ногти / Фитнес / Барбер. Мах 5. Кнопка «Далее» |
| 2 | **Как далеко готов(а) ехать?** | Слайдер 1-30 км. Подсказка: «Большинство выбирают 10-15 км». Кнопка «Далее» |
| 3 | **Когда обычно?** | Выбор времени: Утро (8-12) / День (12-17) / Вечер (17-22). Можно несколько. Кнопка «Начать» → `/app/feed` |

### 4.2 Онбординг бизнеса (`/onboarding/business`)

4 шага.

| Шаг | Название | Элементы |
|---|---|---|
| 1 | **Подключение YClients** | Инструкция: «Войдите в YClients → Настройки → API → Скопируйте ключ». Поле ввода API-ключа + Company ID. Кнопка «Проверить подключение». Статус: ✅ Подключено / ❌ Ошибка |
| 2 | **Выберите тариф** | Карточки трёх тарифов (Start / Growth / Maximum). Выбранный подсвечен. Счётчик: «Ваш радиус: 10 км. Скидка: 10-35%. Охват: ~800 000 человек». Кнопка «Далее» |
| 3 | **Выберите услуги** | Список услуг подгружается из YClients. Выбор галочками. Для каждой: категория, базовая цена, длительность. Кнопка «Далее» |
| 4 | **Готово** | «Ваши слоты появятся в ленте клиентов в течение 15 минут.» Кнопка «В кабинет» → `/business/dashboard` |

## 5. Лента предложений (`/app/feed`)

### Верхняя панель

| Элемент | Функция |
|---|---|
| Лого / название | Навигация домой |
| Текущая геолокация | «📍 Москва, Арбат» |
| Кнопка обновления | Pull-to-refresh (счётчик: осталось N из M) |
| Иконка профиля | → `/app/profile` |
| Иконка броней | → `/app/bookings` (бейдж: кол-во активных) |

### Фильтры (над лентой)

| Фильтр | Тип | Значения |
|---|---|---|
| Категория | Выпадающий (мульти) | Из онбординга + все доступные |
| Расстояние | Слайдер | 1-30 км |
| Скидка от | Слайдер | 0-50% |
| Время | Кнопки | Сегодня / Завтра / +12ч / +6ч |
| Тип услуги | Переключатель | Индивидуальные / Групповые |

Счётчик: «Найдено N предложений»

### Карточка предложения

Каждая карточка содержит:

| Элемент | Описание |
|---|---|
| **Изображение** | Фото студии или услуги |
| **Название студии** | Текст + рейтинг (⭐ 4.8) |
| **Услуга** | «Тайский массаж 60 мин» |
| **Время** | «Сегодня, 18:00–19:00» |
| **Базовая цена** | ~~5 000₽~~ (зачёркнута) |
| **Цена со скидкой** | **3 550₽** (зелёный, крупный) |
| **Скидка** | «−29%» (бейдж) |
| **Расстояние** | «📍 10 км от вас» |
| **Мест осталось** | «1 из 6» (если групповое) |
| **Кнопка** | «Забронировать» |

### Состояния карточки

| Состояние | Внешний вид |
|---|---|
| Доступно | Зелёная кнопка «Забронировать» |
| Мало мест (≤2) | Оранжевый бейдж «2 места» |
| Забронировано (соседом) | Серая, «Только что забрали» + альтернативы |
| Последнее место | Красный бейдж «Последнее» |
| Истекает (<2ч) | Иконка 🔥 «Успей» |

### Действия с карточкой

| Действие | Куда |
|---|---|
| Тап на карточку | `/app/card/{id}` |
| «Забронировать» | Модальное окно подтверждения |
| «❤️» | Добавить в избранное |
| Свайп вправо | В избранное |
| Свайп влево | Скрыть (не показывать сегодня) |

## 6. Детальная карточка (`/app/card/{id}`)

### Содержание

| Секция | Элементы |
|---|---|
| **Фото** | Галерея студии (свайп) |
| **Основное** | Название, категория, адрес на карте, рейтинг, кол-во броней через платформу |
| **Услуга** | Название, длительность, описание |
| **Цена** | Базовая (зачёркнута), цена со скидкой, размер скидки, формула расчёта |
| **Время** | Дата, время начала, окончания |
| **Места** | «Свободно X из Y» (если групповое) |
| **Студия** | Краткая информация, ссылка «Все предложения студии» |
| **Кнопка** | «Забронировать за X ₽» |

### Блок с расшифровкой скидки

«Почему такая цена?»

| Строка | Значение |
|---|---|
| Базовая цена | 5 000₽ |
| Расстояние | 10 км из 25 км (40%) |
| Ваша скидка | 15% (база) + 14% (за расстояние) = 29% |
| Итого | **3 550₽** |

## 8. Мои бронирования (`/app/bookings`)

### Табы

| Таб | Содержание |
|---|---|
| **Активные** | Брони на сегодня/завтра со статусом PAID, CONFIRMED |
| **История** | Завершённые (COMPLETED, CANCELLED, LATE_CANCEL) |
| **Ожидают** | HELD (таймер до автоотмены) |

### Карточка брони

| Элемент | Описание |
|---|---|
| Статус | Бейдж: Ожидает / Подтверждено / Завершено / Отменено |
| Студия | Название, адрес |
| Услуга | Название, длительность |
| Дата и время | Форматированная дата |
| Цена | Итоговая цена |
| Таймер (HELD) | «Слот освободится через 4:32» |

### Действия

| Статус | Доступные действия |
|---|---|
| HELD | «Подтвердить», «Отменить» |
| PAID / CONFIRMED (>30 мин до начала) | «Отменить» |
| PAID / CONFIRMED (≤30 мин) | Без действий |
| COMPLETED | «Оставить отзыв», «Повторить» |

## 13. Переходы и состояния

### Основной пользовательский флоу

```
Лендинг → Регистрация → Онбординг → Лента → Карточка → Бронь → Оплата → Успех → Мои брони
```

### Альтернативные пути

```
Лента (нет предложений) → Фильтры (расширить радиус) → Лента
Карточка → Слот заняли → Альтернативы → Бронь → Успех
Лента → Избранное → Лента (фильтр по студии) → Бронь
Мои брони → Отменить → Возврат (статус CANCELLED)
Профиль → Подписка → Апгрейд → Оплата → Новый тариф
Профиль → Предпочтения → Изменить → Лента (обновлённая)
```

### Guard conditions

| Условие | Действие |
|---|---|
| Не авторизован → `/app/*` | Редирект на `/login` |
| Авторизован → `/login` | Редирект на `/app/feed` |
| Не пройден онбординг → `/app/feed` | Редирект на `/onboarding/user` |
| Free-тариф, 3/3 рефреша → `/app/feed` | Кнопка обновления заблокирована, подсказка «Апгрейд» |
| Бизнес не подключил YClients → `/business/services` | Баннер «Подключите YClients» + ссылка на настройки |
